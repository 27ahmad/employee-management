# Employee Management System

## Overview
The Employee Management System is a web application designed to manage employee data. It allows users to create, read, update, and delete employee records. The application is built using a modern tech stack including React, Material-UI, AG Grid, Node.js, Express, and MongoDB.

## Features
- Add new employees
- Edit existing employee details
- Delete employees
- View employee list in a grid with sorting, filtering, and pagination
- Responsive design with Material-UI components
- Toast notifications for user feedback

## Tech Stack
- **Frontend**: React, Material-UI, AG Grid, Axios, React Router, React Toastify
- **Backend**: Node.js, Express, Mongoose
- **Database**: MongoDB
- **Build Tool**: Vite
